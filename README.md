---

- Create, Read, Update, Delete movies

---

## Stack

---

- MongoDB
- Express
- React
- Node JS

- Backend
  - Babel
  - mongoose
  - dotenv
  - nodemon
- Frontend
  - create-react-app
  - react-router-dom
  - Bootstrap
  - style-components
  - axios
  - react-table

---

## Set up Api server

---

- Babel set up
- Express server
- Express server connect with MongoDB
- REST API create movie, get movies, get movie, update movie, delete movie
- Confirm at Postman

```
(Windows cmd, powershell)
cd server
npm start
```
